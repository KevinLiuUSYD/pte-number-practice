# PTE Number Practice Tool

一个用于练习PTE数字发音的交互式网页工具。

## 功能特点

- 🔢 随机生成0到10亿之间的数字进行练习
- 🎤 使用浏览器语音识别API进行实时评判
- 🔄 支持重读练习，可以多次尝试同一数字
- 🔊 提供标准读音播放功能
- ⏱️ 可调节等待时间
- 📊 显示识别置信度作为流畅度参考

## 使用方法

1. 在浏览器中打开 `number prac.html`
2. 点击"开始练习"按钮
3. 允许浏览器访问麦克风权限（只需授权一次）
4. 等待倒计时结束后朗读显示的数字
5. 查看反馈结果，可以点击"重读练习"重新尝试

## 技术说明

- 使用 Web Speech API 进行语音识别
- 支持对数分布随机数生成，确保不同数量级的数字均匀出现
- 持续识别模式，避免重复请求麦克风权限

## 浏览器要求

- Google Chrome（推荐）
- Microsoft Edge
- 其他支持 Web Speech API 的现代浏览器
