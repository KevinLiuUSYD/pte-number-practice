# PTE Number Practice Tool

一个用于练习PTE数字发音的交互式网页工具，帮助提高英语数字的发音准确性。

## 功能特点

- 🔢 **智能随机数字**：使用对数分布生成0到10亿之间的数字，确保不同数量级的数字均匀出现
- 🎤 **实时语音识别**：使用浏览器语音识别API进行实时评判
- ✅ **结果标记**：历史记录中自动标记正确/错误的数字
- 🔄 **重读练习**：支持对同一数字多次练习，直到满意为止
- 📚 **历史记录**：自动记录最近20个练习过的数字，可随时重新练习
- 🔊 **标准读音**：提供标准读音播放功能，帮助学习正确发音
- ⏱️ **可调节等待时间**：支持0-10秒倒计时，方便准备
- 📊 **识别置信度**：显示系统识别置信度作为流畅度参考
- 🎨 **响应式设计**：支持桌面、平板设备（手机端有已知问题）

## 使用方法

### 基本使用流程

1. **打开网页**
   - 访问：`https://kevinliuusyd.github.io/pte-number-practice/`
   - 或直接打开 `index.html` 文件

2. **开始练习**
   - 点击"开始练习"按钮
   - 系统会随机生成一个数字并显示

3. **授权麦克风**
   - 首次使用时，浏览器会请求麦克风权限
   - 点击"允许"授权（只需授权一次）
   - 授权后，后续使用不会再重复请求

4. **等待倒计时**
   - 系统会显示倒计时（默认5秒，可在设置中调节0-10秒）
   - 可以点击"直接读 (Skip)"跳过倒计时
   - 倒计时结束后，系统开始聆听

5. **朗读数字**
   - 清晰、准确地朗读屏幕上显示的数字
   - 系统会自动识别你的发音

6. **查看反馈**
   - 系统会显示识别结果和评判
   - ✅ 绿色边框：读法正确
   - ❌ 红色边框：需要改进
   - 显示识别置信度（流畅度参考）

7. **继续练习**
   - 点击"🔄 重读练习"：重新练习当前数字
   - 点击"开始练习"：练习新的随机数字
   - 点击历史记录中的数字：重新练习历史数字

### 高级功能

#### 历史记录
- 自动记录最近20个练习过的数字
- 点击"📚 历史记录"按钮显示/隐藏历史
- 历史记录中：
  - ✅ 绿色左边框：读对的数字
  - ❌ 红色左边框+浅红背景：读错的数字
  - 无标记：尚未练习或结果未知
- 点击历史记录中的任意数字可重新练习

#### 标准读音
- 在反馈区域点击"🔊 听正确读音"按钮
- 系统会播放该数字的标准英文读音
- 帮助学习正确的发音方式

#### 等待时间设置
- 在页面顶部可以调节等待时间（0-10秒）
- 设置为0时，点击"开始练习"后立即开始聆听
- 适合不同练习节奏的需求

## 浏览器要求

### 推荐浏览器（桌面端）
- ✅ **Google Chrome**（最佳支持）
- ✅ **Microsoft Edge**（基于Chromium）
- ✅ **其他基于Chromium的浏览器**

### 移动端支持情况

⚠️ **重要提示：手机端暂时有已知问题，无法正常使用**

- ❌ **手机Chrome**：存在识别卡死问题，暂不支持
- ❌ **手机Safari**：不支持Web Speech API
- ❌ **其他手机浏览器**：可能存在兼容性问题

**建议使用桌面浏览器进行练习。**

### 为什么手机端有问题？

手机端Chrome浏览器的Web Speech API实现与桌面端不同，存在以下问题：
- 识别启动后可能卡在"正在聆听"状态
- 识别结果可能无法正确返回
- 需要手动停止才能退出识别状态

我们正在努力修复这些问题，但暂时建议使用桌面浏览器。

## 技术说明

- **语音识别**：使用 Web Speech API（webkitSpeechRecognition）
- **数字生成**：对数分布算法，确保不同数量级均匀出现
- **识别模式**：
  - 桌面端：持续识别模式，避免重复请求权限
  - 移动端：单次识别模式（但存在已知问题）
- **数据存储**：所有数据存储在浏览器本地，不会上传到服务器

## 常见问题

### Q: 为什么需要麦克风权限？
A: 语音识别功能需要访问麦克风才能识别你的发音。权限只需授权一次，后续使用不会再提示。

### Q: 识别不准确怎么办？
A: 
- 确保环境安静，减少背景噪音
- 清晰、准确地发音
- 可以多次重读练习，提高准确性
- 使用"听正确读音"功能学习标准发音

### Q: 可以离线使用吗？
A: 可以！下载 `index.html` 文件后，即使没有网络也可以使用。但首次加载可能需要网络连接。

### Q: 历史记录会保存吗？
A: 历史记录存储在浏览器本地，刷新页面后会清空。这是为了确保每次练习都是新的开始。

### Q: 为什么手机端不能用？
A: 手机端Chrome的Web Speech API实现存在兼容性问题，导致识别功能无法正常工作。建议使用桌面浏览器。
